232henley

2020

232henley is an interactive project, an attempt to create a playable "memory palace" of my childhood home. Created at the start of the pandemic as an attempt to prevent the fading of memory, though ultimately there is a failure built into the simulation. All objects were originally drawn in dashed-off notebook sketches, and brought into digital space, with digital memory holes and half-memories "preserved" digitally. Created with drawings and in the game engine Unity, with soundtrack composed on the modular synth.
