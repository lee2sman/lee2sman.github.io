Textual Selfie Station

Textual Selfie Station was a code sketch project created at The School For Poetic Computation. Textual Selfie creates a live video feed in ascii text, with the text generated based on a Google search of the participant. It was presented in a final exhibit of The School For Poetic Computation at Orbital, and has been used to generate works presented online and in further exhibitions.

Github
This program is created in Processing. Additional: Javascript, Bash.
