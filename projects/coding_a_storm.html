Coding A Storm

Coding A Storm is a data visualisation and sonification performance project and installation. To date it has been presented in galleries in California and at the Ace Hotel in New York.

In collaboration with atmospheric scientist Campbell Watson, we have developed techniques to sonify weather data. For the most recent instantiation of Coding A Storm, we pulled data from Helen, the nicknamed weather station installed by Watson on the Ace Hotel roof.

The full performance project includes live musical performance and selection of weather events (via data files) that are sonified to music.

One part of the project includes a data viz and sonified web-based project Helen's Clouds, which compresses 4 months of weather data to a 4 minute time period. The background color reflects the temperature. Cloud appearance reflects cloud cover. Raindrops correspond to mesaurable precipitation.

The project is coded in Javascript with P5.JS and JSON/CSV data files from the Helen weather station
